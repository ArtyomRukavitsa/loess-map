# Извлечение научных данных из публикаций — карта разрезов

Интерактивная карта лёссовых/четвертичных разрезов, извлечённых из публикаций
автоматическим пайплайном (статьи Галая 2014 + Величко 1997 + рисунки-разрезы).

**Стек:** Streamlit + pydeck. Данные — `results.xlsx` (66 разрезов).

## Возможности
- Карта с accuracy-кружками (радиус = неопределённость координат).
- Фильтры: регион, тип отложений, источник, радиус точности.
- Клик по точке → карточка записи (поля схемы) + «Records at this coordinate».
- Вкладки: Карта · Таблица (+ скачать CSV) · Статистика.

## Запуск локально
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Деплой
[Streamlit Community Cloud](https://share.streamlit.io) → New app → этот репозиторий → main file `app.py`.

Координаты — геокодинг по населённому пункту; пусто = `ND` (данные не выдумываются).
Демо первых результатов пайплайна; часть полей пока `ND`.
